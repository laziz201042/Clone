## 🌟 Loyiha haqida

**FINDTube** — bu O'zbekiston dasturchilar jamiyati uchun mo'ljallangan ochiq manbali video platforma. Loyiha YouTube'ning eng mashhur xususiyatlaridan ilhom olib, to'liq o'zbek tilida qurilgan va mahalliy foydalanuvchilar ehtiyojlarini inobatga olgan holda ishlab chiqilgan.

Platforma uch tilda — **O'zbekcha**, **Русский** va **English** — ishlaydi va zamonaviy UI/UX standartlariga to'liq javob beradi.

```
🎯 Maqsad     — O'zbek tilidagi birinchi to'laqonli video platforma
👥 Auditoriya  — O'zbekiston va MDH mamlakatlari foydalanuvchilari
📅 Yaratilgan  — 2024
🔗 Demo        — https://findtub.netlify.app/
```

## 🚀 Loyihani ishga tushirish

### Talablar
- Zamonaviy veb-brauzer (Chrome, Firefox, Safari, Edge)
- Git (klonlash uchun)

### O'rnatish

```bash
# 1. Repozitoriyani klonlash
git clone https://github.com/your-username/findtube.git

# 2. Loyiha papkasiga kirish
cd findtube

# 3. index.html ni brauzerda ochish
# macOS:
open index.html

# Windows:
start index.html

# Linux:
xdg-open index.html
```

> ✅ **Eslatma:** FINDTube sof frontend loyiha — hech qanday server yoki backend kerak emas!

### Netlify orqali deploy qilish

```bash
# 1. Netlify CLI o'rnatish
npm install -g netlify-cli

# 2. Login qilish
netlify login

# 3. Deploy qilish
netlify deploy --prod
```

## ❓ Ko'p so'raladigan savollar

<details>
<summary><b>🎥 Video qanday yuklanadi?</b></summary>

> Navbar yuqorisidagi 📷 kamera belgisini bosing, video faylingizni tanlang va yuklash tugmasini bosing.

</details>

<details>
<summary><b>📺 Kanal ochish mumkinmi?</b></summary>

> Ha! Profil rasmingizni bosing → "Kanal yaratish" tugmasini tanlang → Kanal nomini kiriting va tasdiqlang.

</details>

<details>
<summary><b>🔕 Obunani qanday bekor qilish mumkin?</b></summary>

> Kanal sahifasiga kiring → "Obunada" tugmasini bosing → Bekor qilishni tasdiqlang.

</details>

<details>
<summary><b>🗑️ Ko'rish tarixi o'chiriladimi?</b></summary>

> Ha. Tarix sahifasiga o'ting → "Tozalash" tugmasini bosing → Barcha tarix o'chib ketadi.

</details>

<details>
<summary><b>🌐 Sayt qaysi tillarda ishlaydi?</b></summary>

> FINDTube uchta tilda ishlaydi: 🇺🇿 O'zbekcha, 🇷🇺 Русский, 🇬🇧 English. Sozlamalar orqali o'zgartirishingiz mumkin.

</details>
